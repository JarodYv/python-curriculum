# 简介

这个项目通过创建一个颜色名与颜色值一一对应的字典来介绍字典。然后从字典中找到颜色值并用其创建一张色彩斑斓的海报。

# 学习目标

+ 字典 - 创建并查找值;
+ 乌龟图形 - 文本，字体和颜色;

# 挑战

+ 更多颜色! - 使用取色网站找到更多颜色的十六进制颜色值并将他们加入字典。

+ 创建海报 - 自定义一个颜色板字典并用它来创建一张海报。

# 常见问题

+ 孩子们需要时常提醒每一个字典元素后要加上逗号 ',' 。
